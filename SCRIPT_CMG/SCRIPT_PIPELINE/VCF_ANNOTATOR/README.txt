VCF ANNOTATOR

Author: Matteo Di Giovannantonio
Contact: matteodeg@gmail.com

DESCRIPTION:
VCF annotator lavora su un file vcf in input, annotato con VEP, per aggiungere dati estratti da database estratti. Con VCF ANNOTATOR puoi inserire le features dei seguenti database:

-CLINVAR (formato vcf)
-HUMSAVAR (tab delimited)
-ESP Exome Variant Server (formato vcf)
-GERP score ()
